>Its starts with Elysia and ends with Elysia

![elysia](https://github.com/user-attachments/assets/d1f90f42-20b3-4df8-862b-f8c5a4f94f1a)

## Development
To start the development server run:
```bash
bun install
bun dev
```

## Limitations

**No Database**: This project does not have any databases. It uses [pokemon-tcg-pocket-cards](https://github.com/chase-manning/pokemon-tcg-pocket-cards) as data source.
